## Description

**Quand les Yeux se Croisent** est une collection de regards humains et animaux présentée dans une installation immersive. En observant les écrans, le visiteur découvre que son propre regard s’ajoute à la collection. Le dialogue visuel entre humains et animaux met en lumière le lien profond qui unit tous les êtres vivants, créant un pont sensible entre le visiteur, les autres et les images qui l’entourent.
